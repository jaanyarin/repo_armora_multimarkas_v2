# F2-019 - Crear Mapa de Rutas

## Objetivo

Registrar el flujo de implementacion para la pantalla Crear Mapa de Rutas.

## Prototipo

Se versiono un index funcional en la carpeta de prototipos.

## Alcance UX

- Seleccionar zona comercial.
- Seleccionar rutas activas.
- Dibujar poligono de cobertura.
- Validar datos antes de guardar.
- Mostrar GeoJSON generado.

## Integracion futura

La pantalla final debe vivir en el frontend web y conectarse a los contratos existentes de zonas y rutas.
